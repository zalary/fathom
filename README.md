# Fathom

A dark, muted, low-contrast color theme for the deep. Calm. Focused. No eye strain.

Inspired by the deep ocean — desaturated blue-greens, silver, muted olive. Named after the unit of ocean depth.

## Ports

- [Ghostty](ghostty/)

## Install

### Ghostty

Copy `ghostty/fathom` to your Ghostty themes directory:

```sh
cp ghostty/fathom ~/.config/ghostty/themes/fathom
```

Then in your Ghostty config:

```
theme = fathom
```
